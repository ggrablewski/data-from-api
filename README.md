# data-from-api
Project for retrieving data from API

## Contents

- This project reads from [https://jsonplaceholder.typicode.com/](https://jsonplaceholder.typicode.com/posts) via API.
- The contents read consists of posts.
- Each post is saved in JSON format in a separate file with the name: <post_id>.json - in the respective folder: /posts_ja or /posts_java.
